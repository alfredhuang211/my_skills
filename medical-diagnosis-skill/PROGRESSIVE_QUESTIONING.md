# 渐进式问诊设计方案

## 设计原则

### 核心理念
- **逐步深入**：不一次性抛出所有问题
- **问题分组**：2-3个强相关问题为一组
- **动态决策**：根据回答决定下一步问什么
- **用户友好**：使用选择按钮，减少输入负担

### 问诊节奏
```
基础信息（2题） → 鉴别诊断（分3轮，每轮2-3题） → TNSS评分（分2轮，每轮2题） 
→ 病程评估（1-2题） → 特殊情况（按需，每轮2-3题） → 生成方案
```

---

## 阶段1：初步症状识别（2个问题）

### 问题组1.1：核心症状 + 病程
```
触发：用户提到"打喷嚏""鼻炎""鼻痒""流鼻涕""鼻塞"等关键词

【问题1】您主要有哪些不舒服的症状？（多选）
□ 打喷嚏
□ 流鼻涕
□ 鼻痒
□ 鼻塞
□ 咽痛
□ 其他（请描述）

【问题2】这些症状持续多久了？
□ 1-3天（刚开始）
□ 4-7天（不到一周）
□ 1-2周
□ 2-4周
□ 超过1个月
□ 反复发作很久了
```

**AI决策点1**：
- 如果病程 ≤7天 → 进入鉴别诊断（重点排除感冒）
- 如果病程 >7天且反复 → 跳过感冒排除，继续鉴别鼻窦炎/血管运动性鼻炎

## 阶段2：鉴别诊断（分3轮，渐进式排除）

### 第1轮：排除感冒（3个问题，仅当病程≤7天时询问）

```
【问题3】您是否有以下症状？（多选）
□ 发热（体温>37.5℃）
□ 咽喉疼痛
□ 全身酸痛、乏力
□ 都没有

【问题4】鼻涕的颜色和性状是？
□ 清水样（透明）
□ 黄色或绿色（浓稠）
□ 从清涕转为黄涕
□ 不确定

【问题5】症状是否在逐渐好转？
□ 是，感觉一天比一天好
□ 否，一直没好转或更严重了
```

**AI判断**：
- 如果：发热 + 咽痛 + 黄涕 + 症状好转 → **疑似感冒**
  - 输出建议："根据您的症状，更像是感冒。建议多休息、多喝水，症状通常7-10天内自然缓解。"
  - **结束问诊**（不继续）

- 如果：无发热 + 清涕 + 症状未好转 → **排除感冒**
  - 继续下一轮排除

---

### 第2轮：排除鼻窦炎（3个问题）

```
【问题6】您的鼻涕是否为脓性（黄色或绿色）？
□ 是，黄色或绿色的浓稠鼻涕
□ 否，清水样透明鼻涕
□ 有时清有时黄

【问题7】是否有以下症状？（多选）
□ 面颊部或额头疼痛/压迫感
□ 嗅觉明显下降或闻不到气味
□ 头痛（尤其是低头时加重）
□ 都没有

【问题8】目前是否有发热？
□ 是（体温>37.5℃）
□ 否
```

**AI判断**：
- 如果：脓性鼻涕 + (面部痛 OR 嗅觉降 OR 头痛) → **疑似鼻窦炎**
  - 输出建议："您的症状提示可能为鼻窦炎，建议前往耳鼻喉科就诊，可能需要抗生素治疗。"
  - **结束问诊**

- 如果：清涕 + 无面部痛 + 无嗅觉降 → **排除鼻窦炎**
  - 继续下一轮排除

---

### 第3轮：排除血管运动性鼻炎（2个问题）

```
【问题9】什么情况下症状会明显加重？（多选）
□ 接触冷空气
□ 闻到刺激性气味（香水、油烟等）
□ 温度骤变
□ 体位变化（躺下、起床）
□ 接触花粉、尘螨、宠物
□ 不明确

【问题10】您的鼻痒症状明显吗？
□ 非常痒，经常忍不住揉鼻子
□ 有点痒，但不严重
□ 基本不痒
```

**AI判断**：
- 如果：冷空气/气味触发 + 无明显鼻痒 + 无过敏原 → **疑似血管运动性鼻炎**
  - 输出建议："您的症状可能是血管运动性鼻炎，建议避免诱因，可使用鼻用激素或生理盐水冲洗。"
  - **结束问诊**

- 如果：过敏原触发 + 明显鼻痒 → **确认过敏性鼻炎**
  - 提示："已排除感冒、鼻窦炎、血管运动性鼻炎，您的症状符合过敏性鼻炎特征。接下来我需要评估症状的严重程度。"
  - **进入TNSS评分**

## 阶段3：TNSS评分（分2轮，每轮2个症状）

### 第1轮：打喷嚏 + 流鼻涕

```
【问题11】您每天大概打多少个喷嚏？
□ 0分：不打喷嚏或很少
□ 1分：1-5个/天（轻度，不影响生活）
□ 2分：6-10个/天（中度，明显不适）
□ 3分：11个以上/天（重度，连续打喷嚏，严重影响生活）

【问题12】流鼻涕的程度如何？
□ 0分：不流鼻涕
□ 1分：少量，每天擦拭≤5次
□ 2分：明显，每天擦拭6-10次
□ 3分：大量，每天擦拭≥11次或持续流淌
```

---

### 第2轮：鼻痒 + 鼻塞

```
【问题13】鼻子痒的程度如何？
□ 0分：不痒
□ 1分：偶尔痒，每天揉鼻≤3次
□ 2分：经常痒，每天揉鼻4-10次
□ 3分：持续难忍，每天揉鼻≥11次，严重影响注意力

【问题14】鼻塞的情况如何？
□ 0分：不堵
□ 1分：偶尔堵，主要在夜间/晨起
□ 2分：白天也堵，交替性鼻塞，偶尔张口呼吸
□ 3分：双侧持续鼻塞，只能用嘴呼吸
```

**AI自动计算**：
- TNSS总分 = Q11 + Q12 + Q13 + Q14
- 输出："您的TNSS评分为：__分"

---

## 阶段4：病程评估（1-2个问题）

### 问题组4.1：发作频率 + 持续时间

```
【问题15】这些症状每周发作几天？
□ 少于4天/周（间歇性发作）
□ 4天或以上/周（几乎每天都有）

【问题16】症状已经持续多久了？
□ 不到4周（短期发作）
□ 4周或更长时间（长期反复）
```

**AI决策**：
- 根据Q15+Q16判断：间歇性/持续性
- 输出分型："您的病程为：**持续性过敏性鼻炎**"

---

## 阶段5：特殊情况筛查（按需询问，分组提问）

### 触发条件判断
```
AI先快速判断是否需要询问特殊情况：
- 如果TNSS<5分（轻度）且无复杂症状 → 跳过，直接生成方案
- 如果TNSS≥5分或准备推荐用药 → 必须询问特殊情况
```

---

### 第1组：基本信息（2个问题）

```
【问题17】请告诉我您的年龄范围？
□ 儿童（<12岁，请注明年龄：___）
□ 成人（12-65岁）
□ 老年人（≥65岁）

【问题18】您是否属于以下特殊人群？（单选）
□ 孕妇（请注明孕周：___）
□ 哺乳期
□ 都不是
```

**AI决策**：
- 如果：孕妇/哺乳期 → 进入孕妇专用药物方案（跳过部分后续问题）
- 如果：儿童 → 进入儿童用药方案
- 如果：成人 → 继续询问基础疾病

□ 都没有

```

### 第4组：生活习惯（1个问题，仅在推荐含伪麻黄碱药物时询问）

- 如果经常饮酒 → 不推荐含伪麻黄碱的复方制剂

```
AI后台处理：
1. 综合TNSS评分 + 病程分型 → 确定基础方案
2. 结合特殊人群 + 合并疾病 → 调整药物选择
3. 自动排除禁忌药物
4. 生成个性化方案

输出格式：

   - 注意：需连续规律使用

## 问诊总结：问题数量统计

| 阶段 | 问题数量 | 说明 |
|------|---------|------|
| 初步症状识别 | 2题 | 必问 |

| 病程评估 | 2题 | 必问 |
| 特殊情况-基本信息 | 2题 | TNSS≥5分时必问 |
| 特殊情况-合并疾病 | 2题 | 非孕妇时询问 |

| 特殊情况-饮酒 | 1题 | 推荐伪麻黄碱时询问 |
| **最少问题数** | **10题** | 轻度AR，无复杂情况 |
| **常规问题数** | **15-18题** | 中度AR，有特殊情况 |
| **最多问题数** | **23题** | 重度AR，全部流程 |

## 关键优化点总结

### ✅ 改进前的问题
- 一次性询问10+个问题，用户体验差
- 所有信息平铺，没有逻辑层次
- 不分场景，全部用户都问相同问题

### ✨ 改进后的优势

1. **渐进式节奏**
   - 2-3个问题为一组
   - 每组问题强相关
   - 逐步深入，不唐突

2. **智能决策树**
   - 根据病程决定是否问感冒相关问题
   - 根据TNSS分数决定是否深入询问特殊情况
   - 根据人群特征调整问题重点

3. **最小化问题数**
   - 轻度患者：只问10题
   - 中度患者：问15-18题
   - 复杂病例：最多23题

4. **用户体验提升**
   - 每次只需回答2-3个问题
   - 进度透明（显示"第X/总数题"）
   - 每轮结束有阶段性反馈

5. **临床逻辑清晰**
   - 先排除其他疾病（鉴别诊断）
   - 再评估症状严重程度（TNSS）
   - 最后筛查特殊情况（用药安全）
   - 符合医生真实问诊思路
