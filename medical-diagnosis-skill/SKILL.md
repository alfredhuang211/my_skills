# Medical Diagnosis Assistant - 医疗诊断助手

## Overview

智能医疗诊断助手，专注于过敏性鼻炎（Allergic Rhinitis, AR）的AI临床诊断与用药指导。基于临床指南，通过结构化问诊、症状评分、分型分度，自动生成个性化治疗方案。

## Skill Activation Triggers

此技能应在以下情况自动激活：

### 关键症状触发词
- **鼻部症状**：打喷嚏、流鼻涕、清水涕、鼻痒、鼻塞、鼻子堵、鼻子痒
- **相关症状**：过敏、鼻炎、鼻敏感、季节性鼻炎
- **询问用药**：鼻炎吃什么药、过敏性鼻炎怎么治疗、鼻炎用药

### 场景触发
- 用户描述2个或以上的鼻部症状（打喷嚏、流涕、鼻痒、鼻塞）
- 用户提到过敏史或季节性发作
- 用户询问鼻炎相关治疗建议

## Core Clinical Logic - 核心临床逻辑

### 1. 识别主诉
- **典型症状**：打喷嚏、清水涕、鼻痒、鼻塞（≥2个 + 反复发作）
- **必要条件**：症状反复发作，非一次性

### 2. 鉴别诊断排除流程（必须优先执行）

**⚠️ 重要：在确诊过敏性鼻炎前，必须逐步排除以下疾病！**

这是诊断的第一步，必须在进行TNSS评分和治疗方案制定之前完成。

#### 排除步骤：

**第一步：排除感冒（急性上呼吸道感染）**
- 询问关键问题：
  - 症状是否在7-10天内自然缓解？
  - 是否伴有咽痛、发热（>37.5℃）？
  - 鼻涕是否从清涕转为黄涕/绿涕？
  - 是否有全身乏力、肌肉酸痛？
- 鉴别要点：
  - ✅ 感冒：病程短（7-10天），症状会自行好转
  - ❌ 过敏性鼻炎：症状反复发作，持续时间长
- 如果符合感冒特征 → 提示："根据您的症状，更像是感冒。建议多休息、多喝水，症状通常7-10天内自然缓解。如持续不愈或加重，请就医。"

**第二步：排除鼻窦炎（急性/慢性）**
- 询问关键问题：
  - 鼻涕是否为黄色或绿色脓性分泌物？
  - 是否有面颊部疼痛或压迫感？
  - 是否有嗅觉明显下降或丧失？
  - 是否伴有发热？
  - 是否有头痛（前额或面部）？
- 鉴别要点：
  - ✅ 鼻窦炎：脓性鼻涕 + 面部疼痛 + 嗅觉下降
  - ❌ 过敏性鼻炎：清水样鼻涕 + 鼻痒明显
- 如果符合鼻窦炎特征 → 提示："您的症状提示可能为鼻窦炎，需要医生评估是否需要抗生素治疗。建议前往耳鼻喉科就诊。"

**第三步：排除血管运动性鼻炎**
- 询问关键问题：
  - 主要诱因是什么？（冷空气、刺激性气味、温度变化、体位变化）
  - 鼻痒是否明显？
  - 是否有明确的过敏原（花粉、尘螨、宠物）？
  - 打喷嚏是否频繁？
- 鉴别要点：
  - ✅ 血管运动性鼻炎：冷空气/气味触发 + 无明显鼻痒 + 无明确过敏原
  - ❌ 过敏性鼻炎：明显鼻痒 + 打喷嚏频繁 + 有明确过敏原
- 如果符合血管运动性鼻炎特征 → 提示："您的症状可能是血管运动性鼻炎，建议避免诱因（冷空气、刺激性气味等），必要时可使用鼻用激素。"

**第四步：确认过敏性鼻炎**
只有在排除上述三种疾病后，才能进入过敏性鼻炎的诊断流程：
- ✅ 典型症状：打喷嚏 + 清水涕 + 鼻痒 + 鼻塞（≥2个）
- ✅ 反复发作，非一次性
- ✅ 有明确诱因（花粉、尘螨、宠物等）
- ✅ 鼻痒明显

**AI工作流程调整：**
```
用户输入症状 
  ↓
【第一步】逐步排除：感冒 → 鼻窦炎 → 血管运动性鼻炎
  ↓

  ↓
【第三步】进入TNSS评分和问诊树
  ↓

```

### 3. 结构化问诊树

#### 3.1 核心症状评估（TNSS评分系统）

**打喷嚏（Sneeze）**
- [ ] 0分：无
- [ ] 1分：1-5个/天（轻度，不影响生活）
- [ ] 2分：6-10个/天（中度，明显不适，轻度影响睡眠/工作）
- [ ] 3分：≥11个/天（重度，阵发性连打，严重影响睡眠、生活、注意力）

**流涕（Rhinorrhea）**
- [ ] 0分：无
- [ ] 1分：少量，≤5次擦拭/天（轻度，不往外流）
- [ ] 2分：明显，6-10次擦拭/天（中度，轻度影响生活）
- [ ] 3分：大量，≥11次或持续流淌（重度，严重影响睡眠/工作）

**鼻痒（Nasal Itch）**
- [ ] 0分：无
- [ ] 1分：偶尔痒，≤3次揉鼻/天（轻度，不影响生活）
- [ ] 2分：经常痒，4-10次揉鼻/天（中度，轻度影响注意力）
- [ ] 3分：持续难忍，≥11次揉鼻/天（重度，严重影响睡眠、工作、情绪）

## Diagnostic Algorithm - 诊断算法

### Step 1: TNSS评分计算
```
TNSS总分 = 打喷嚏分数 + 流涕分数 + 鼻痒分数 + 鼻塞分数
范围：0-12分
```

### Step 2: 分型（按病程）

| 分型 | 诊断标准 |
|------|---------|
| **间歇性** | 症状 <4天/周 或 <连续4周 |
| **持续性** | 症状 ≥4天/周 且 ≥连续4周 |

### Step 3: 分度（按严重程度）

| 分度 | TNSS评分 | 临床表现 |
|------|---------|---------|
| **轻度** | 0-4分 | 不影响睡眠、不影响生活 |
| **中度** | 5-8分 | 明显不适，轻度影响生活/睡眠 |
| **重度** | 9-12分 | 严重影响睡眠、工作、学习 |

## Treatment Protocol - 治疗方案

### 一线用药

#### 1. 鼻用糖皮质激素（首选）
- **糠酸莫米松鼻喷剂**
- **丙酸氟替卡松鼻喷剂**
- **布地奈德鼻喷剂**

**适用**：中重度、持续性首选  
**疗程**：至少2-4周，不能症状一好就停  
**注意**：需连续规律使用才能达到最佳效果

#### 2. 第二代口服抗组胺药
- **氯雷他定**
- **地氯雷他定**
- **西替利嗪**
- **左西替利嗪**

**适用**：轻中度、发作初期、鼻痒明显  
**特点**：起效快，但对鼻塞效果弱  
**优势**：无嗜睡、无依赖性

#### 3. 鼻用抗组胺药
- **盐酸氮卓斯汀鼻喷剂**
- **盐酸左卡巴斯汀鼻喷剂**

**特点**：起效快，适合临时控制

### 二线联合用药

#### 4. 白三烯受体拮抗剂
- **孟鲁司特钠**

**适用**：合并哮喘、咳嗽、儿童

#### 5. 鼻腔冲洗
- **生理盐水/海盐水**

**适用**：全人群、孕妇、儿童首选辅助  
**作用**：物理清除过敏原、缓解鼻塞

#### 6. 肥大细胞膜稳定剂
- **色甘酸钠**

**适用**：预防、儿童、轻症

## Safety Rules - 用药安全规则

### ⚠️ 绝对禁忌与警告

#### 1. 鼻用减充血剂警告
- **药物**：羟甲唑啉、萘甲唑啉
- **限制**：连续使用不超过3-5天
- **原因**：长期使用导致药物性鼻炎（反弹性鼻塞）

#### 2. 第一代抗组胺药慎用
- **药物**：扑尔敏、苯海拉明等
- **副作用**：嗜睡、乏力
- **警告**：不建议开车、高空作业
- **禁忌**：青光眼、前列腺增生患者

#### 3. 哮喘患者特殊警告
- 鼻炎控制差→可能诱发哮喘
- **必须提示**：同时管理上下呼吸道
- **首选**：鼻用激素 + 孟鲁司特

#### 4. 特殊人群用药调整

**孕妇**（根据孕周）
- ✅ 相对安全：生理盐水冲洗、氯雷他定、布地奈德鼻喷
- ⚠️ 慎用：西替利嗪
- ❌ 避免：伪麻黄碱、第一代抗组胺

**青光眼患者**
- ❌ 禁用：第一代抗组胺、减充血剂
- ✅ 安全：鼻用激素、二代抗组胺（左西替利嗪）

**前列腺增生患者**
- ❌ 禁用：第一代抗组胺
- ✅ 安全：鼻用激素、二代抗组胺

**肝肾功能不全**
- 需调整剂量或选择代谢途径不同的药物

### 🚨 危险信号（立即就医）
- 哮喘发作
- 呼吸困难
- 明显喘息
- 高热不退
- 严重头痛、面部肿痛（提示鼻窦炎并发症）

## AI Workflow - AI工作流程（渐进式问诊）

### 🎯 核心原则：分组渐进，避免一次性提问

**问诊节奏**：
```
初步症状（2题）→ 鉴别诊断（分3轮，每轮2-3题）→ TNSS评分（分2轮，每轮2题）
→ 病程评估（2题）→ 特殊情况（按需，分组询问）→ 生成方案
```

**问题数量控制**：
- 轻度简单病例：10-12题
- 中度常规病例：15-18题  
- 复杂病例：最多23题

def calculate_tnss(sneeze, rhinorrhea, itch, congestion):
    """
    计算TNSS总分
    参数: 各症状分数 (0-3)
    返回: TNSS总分 (0-12)
    """
    return sneeze + rhinorrhea + itch + congestion

def classify_severity(tnss_score):
    """分度"""
    if tnss_score <= 4:
        return "轻度"
    elif tnss_score <= 8:
        return "中度"
    else:
        return "重度"

def classify_pattern(days_per_week, duration_weeks):
    """分型"""
    if days_per_week < 4 or duration_weeks < 4:
        return "间歇性"
    else:
        return "持续性"
