# Medical Diagnosis Assistant - 医疗诊断助手

## Overview

智能医疗诊断助手，专注于过敏性鼻炎（Allergic Rhinitis, AR）的AI临床诊断与用药指导。基于临床指南，通过结构化问诊、症状评分、分型分度，自动生成个性化治疗方案。

## Skill Activation Triggers

此技能应在以下情况自动激活：

### 关键症状触发词
- **鼻部症状**：打喷嚏、流鼻涕、清水涕、鼻痒、鼻塞、鼻子堵、鼻子痒
- **相关症状**：过敏、鼻炎、鼻敏感、季节性鼻炎
- **询问用药**：鼻炎吃什么药、过敏性鼻炎怎么治疗、鼻炎用药

### 场景触发
- 用户描述2个或以上的鼻部症状（打喷嚏、流涕、鼻痒、鼻塞）
- 用户提到过敏史或季节性发作
- 用户询问鼻炎相关治疗建议

## Core Clinical Logic - 核心临床逻辑

### 1. 识别主诉
- **典型症状**：打喷嚏、清水涕、鼻痒、鼻塞（≥2个 + 反复发作）
- **必要条件**：症状反复发作，非一次性

### 2. 快速排除鉴别诊断
在确诊过敏性鼻炎前，必须排除以下疾病：

| 疾病 | 鉴别要点 |
|------|---------|
| **感冒** | 病程短（7-10天），可能伴咽痛、发热、黄涕 |
| **鼻窦炎** | 黄/绿脓涕、面颊痛、嗅觉下降、发热 |
| **血管运动性鼻炎** | 诱因以冷空气、气味、体位变化为主，无明显鼻痒 |

### 3. 结构化问诊树

#### 3.1 核心症状评估（TNSS评分系统）

**打喷嚏（Sneeze）**
- [ ] 0分：无
- [ ] 1分：1-5个/天（轻度，不影响生活）
- [ ] 2分：6-10个/天（中度，明显不适，轻度影响睡眠/工作）
- [ ] 3分：≥11个/天（重度，阵发性连打，严重影响睡眠、生活、注意力）

**流涕（Rhinorrhea）**
- [ ] 0分：无
- [ ] 1分：少量，≤5次擦拭/天（轻度，不往外流）
- [ ] 2分：明显，6-10次擦拭/天（中度，轻度影响生活）
- [ ] 3分：大量，≥11次或持续流淌（重度，严重影响睡眠/工作）

**鼻痒（Nasal Itch）**
- [ ] 0分：无
- [ ] 1分：偶尔痒，≤3次揉鼻/天（轻度，不影响生活）
- [ ] 2分：经常痒，4-10次揉鼻/天（中度，轻度影响注意力）
- [ ] 3分：持续难忍，≥11次揉鼻/天（重度，严重影响睡眠、工作、情绪）

**鼻塞（Nasal Congestion）**
- [ ] 0分：无堵塞
- [ ] 1分：偶尔堵，主要在夜间/晨起（轻度，能用鼻呼吸）
- [ ] 2分：白天也堵，交替性鼻塞（中度，偶尔张口呼吸）
- [ ] 3分：全天双侧持续鼻塞（重度，必须张口呼吸，严重影响睡眠/嗅觉）

## Diagnostic Algorithm - 诊断算法

### Step 1: TNSS评分计算
```
TNSS总分 = 打喷嚏分数 + 流涕分数 + 鼻痒分数 + 鼻塞分数
范围：0-12分
```

### Step 2: 分型（按病程）

| 分型 | 诊断标准 |
|------|---------|
| **间歇性** | 症状 <4天/周 或 <连续4周 |
| **持续性** | 症状 ≥4天/周 且 ≥连续4周 |

### Step 3: 分度（按严重程度）

| 分度 | TNSS评分 | 临床表现 |
|------|---------|---------|
| **轻度** | 0-4分 | 不影响睡眠、不影响生活 |
| **中度** | 5-8分 | 明显不适，轻度影响生活/睡眠 |
| **重度** | 9-12分 | 严重影响睡眠、工作、学习 |

## Treatment Protocol - 治疗方案

### 一线用药

#### 1. 鼻用糖皮质激素（首选）
- **糠酸莫米松鼻喷剂**
- **丙酸氟替卡松鼻喷剂**
- **布地奈德鼻喷剂**

**适用**：中重度、持续性首选  
**疗程**：至少2-4周，不能症状一好就停  
**注意**：需连续规律使用才能达到最佳效果

#### 2. 第二代口服抗组胺药
- **氯雷他定**
- **地氯雷他定**
- **西替利嗪**
- **左西替利嗪**

**适用**：轻中度、发作初期、鼻痒明显  
**特点**：起效快，但对鼻塞效果弱  
**优势**：无嗜睡、无依赖性

#### 3. 鼻用抗组胺药
- **盐酸氮卓斯汀鼻喷剂**
- **盐酸左卡巴斯汀鼻喷剂**

**特点**：起效快，适合临时控制

### 二线联合用药

#### 4. 白三烯受体拮抗剂
- **孟鲁司特钠**

**适用**：合并哮喘、咳嗽、儿童

#### 5. 鼻腔冲洗
- **生理盐水/海盐水**

**适用**：全人群、孕妇、儿童首选辅助  
**作用**：物理清除过敏原、缓解鼻塞

#### 6. 肥大细胞膜稳定剂
- **色甘酸钠**

**适用**：预防、儿童、轻症

## Safety Rules - 用药安全规则

### ⚠️ 绝对禁忌与警告

#### 1. 鼻用减充血剂警告
- **药物**：羟甲唑啉、萘甲唑啉
- **限制**：连续使用不超过3-5天
- **原因**：长期使用导致药物性鼻炎（反弹性鼻塞）

#### 2. 第一代抗组胺药慎用
- **药物**：扑尔敏、苯海拉明等
- **副作用**：嗜睡、乏力
- **警告**：不建议开车、高空作业
- **禁忌**：青光眼、前列腺增生患者

#### 3. 哮喘患者特殊警告
- 鼻炎控制差→可能诱发哮喘
- **必须提示**：同时管理上下呼吸道
- **首选**：鼻用激素 + 孟鲁司特

#### 4. 特殊人群用药调整

**孕妇**（根据孕周）
- ✅ 相对安全：生理盐水冲洗、氯雷他定、布地奈德鼻喷
- ⚠️ 慎用：西替利嗪
- ❌ 避免：伪麻黄碱、第一代抗组胺

**青光眼患者**
- ❌ 禁用：第一代抗组胺、减充血剂
- ✅ 安全：鼻用激素、二代抗组胺（左西替利嗪）

**前列腺增生患者**
- ❌ 禁用：第一代抗组胺
- ✅ 安全：鼻用激素、二代抗组胺

**肝肾功能不全**
- 需调整剂量或选择代谢途径不同的药物

### 🚨 危险信号（立即就医）
- 哮喘发作
- 呼吸困难
- 明显喘息
- 高热不退
- 严重头痛、面部肿痛（提示鼻窦炎并发症）

## AI Workflow - AI工作流程

### Phase 1: 症状识别与问诊
```
1. 捕获触发词 → 激活技能
2. 友好开场："我注意到您提到了[症状]，为了更好地帮助您，我需要了解一些情况。"
3. 按问诊树逐步收集信息（使用选择按钮，减少用户输入负担）
4. 实时计算TNSS评分
```

### Phase 2: 鉴别诊断
```
1. 根据症状特征排除感冒、鼻窦炎、血管运动性鼻炎
2. 如无法明确诊断 → 提示转人工问诊或线下就医
```

### Phase 3: 分型分度
```
1. 计算TNSS总分
2. 判断病程（间歇性/持续性）
3. 确定严重程度（轻/中/重）
4. 输出诊断结果示例：
   "根据您的症状，初步判断为：**中度持续性过敏性鼻炎**（TNSS评分：7分）"
```

### Phase 4: 生成治疗方案
```
1. 根据分型分度匹配阶梯治疗
2. 结合特殊人群、合并疾病调整用药
3. 自动排除禁忌药物
4. 生成个性化方案（包含：药物、剂量、疗程、注意事项）
```

### Phase 5: 用药指导与购买（可选）
```
1. 询问：是否需要生成用药购买链接？
   - 是 → 收集用户偏好（经济性、品牌、剂型、时效、配送）
   - 否 → 保存记录，结束问诊

2. 生成购买链接（如果有商家接口）
   - 筛选匹配产品（1-2个选项）
   - 展示：价格、品牌、性状、时效、配送时间

3. 购买前提示：
   "如有特殊病情、既往病史、其他用药情况及孕育饮酒等隐瞒，造成不良反应后果自负；
    如用药2-3天症状无改善，请及时线下就医。"
   - 同意 → 继续购买
   - 不同意 → 无法操作
```

### Phase 6: 随访与用药调整
```
在同一问诊记录下，支持用药后随访：

1. 询问：是否咨询用药调整？
   - 否 → 结束
   - 是 → 进入调整流程

2. 收集信息：
   - 用药时间（天数）
   - 症状变化（鼻塞、鼻涕、鼻痒、打喷嚏改善程度）
   - 不良反应（鼻腔疼痛、干燥、出血、嗜睡、乏力等）

3. AI分析：
   - 症状改善 → 减量或调整用药频次
   - 症状无改善 → 调整方案或建议就医
   - 出现不良反应 → 换药或停药

4. 示例调整：
   "您的症状明显改善，建议糠酸莫米松鼻喷剂从每天2次减为每天1次，
    继续使用7天后可重新评估。"
```

def calculate_tnss(sneeze, rhinorrhea, itch, congestion):
    """
    计算TNSS总分
    参数: 各症状分数 (0-3)
    返回: TNSS总分 (0-12)
    """
    return sneeze + rhinorrhea + itch + congestion

def classify_severity(tnss_score):
    """分度"""
    if tnss_score <= 4:
        return "轻度"
    elif tnss_score <= 8:
        return "中度"
    else:
        return "重度"

def classify_pattern(days_per_week, duration_weeks):
    """分型"""
    if days_per_week < 4 or duration_weeks < 4:
        return "间歇性"
    else:
        return "持续性"
